使用说明

一、功能概述

按自定义格式将数据解析成域段，并可将域段反解析为整条数据。

  • 多进制：hex / dec / bin
  • 双模式：Length、Range [高:低]
  • 嵌套域段：子域段解析协议结构
  • 配置：Save / Load JSON

二、操作流程

  1. 填写 Input Data、Bit Width、Base
  2. 配置域段名称、Length 或 Range、模式切换、Base；可添加根域段与子域段
  3. Parse Data 正向解析；修改值后 Reverse Parse 合并

三、注意事项

  • Range 高位 ≥ 低位；根域段总长度勿超过 Bit Width
  • 反向解析时子域段总长度须与父域段 Length 一致（Length 模式）
